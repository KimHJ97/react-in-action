# 프로젝트 생성

## 관련 문서

 - 자바스크립트 프로젝트 생성법 정리: https://daleseo.com/js-npm-init/

<br/>

## npm 명령어

 - __프로젝트 생성 명령어__
    - `npm init xxx`: npm 저장소에섯 `create-xxx` 패키지를 찾아서 실행해준다.
    - `npm create xxx`: npm 저장소에서 `create-xxx` 패키지를 찾아서 실행해준다.
        - `npm init gatsby`, `npm create gatsby`, `npx create-gatsby` 모두 동일하다.
 - __버전 명시하기__
    - 버전을 명시하기 않는 경우 최신 버전으로 설치되지 않고, 해당 컴퓨터에 캐시되어 있는 initializer가 사용될 가능성이 있다.
```bash
# 최신 버전으로 생성
npm init vite@latest 프로젝트명

# 특정 버전으로 생성
npm init vite@6.5.0 프로젝트명
```
