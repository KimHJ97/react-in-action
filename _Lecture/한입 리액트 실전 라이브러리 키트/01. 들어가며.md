# 들어가며

## 1. 실습 환경 설정하기

 - Node.js 설치
    - https://nodejs.org/ko/download
 - VS Code 설치
    - Extension: Prettier, ESLint, Error Lens, 
        - Tailwind CSS IntelliSense: 자동완성, 린팅, 호버 프리뷰 기능 제공

<br/>

## 2. 리액트 앱 생성 및 초기 설정하기

```bash
# 프로젝트 생성
npm create vite@6.5.0 onebite-exam

Select a framework -> React
Select a variant: -> TypeScript

# 프로젝트로 이동 & 의존성 설치
cd onebite-exam
npm install
npm run dev
```

### 초기 생성 파일

 - `tsconfig.app.json`: 브라우저 환경에서만 실행되는 코드들에 대한 타입스크립트 옵션
 - `tsconfig.node.json`: Node.js 환경에서 실행되는 코드들에 대한 타입스크립트 옵션 (예시: 테스트 코드)
 - `tsconfig.json`: 참조만 관리. 각 tsconfig들을 묶어줌.
