# Tailwind CSS

## 1. Tailwind CSS 소개

### 1-1. Tailwind CSS란

- CSS 규칙들을 text-white, font-bold, bg-black 같은 미리 정의된 유틸리티 클래스로 제공하는 도구
- 별도의 CSS 파일 생성이나 규칙 작성 없이 클래스 이름만으로 스타일링 가능
- 기본 제공 유틸리티 클래스가 매우 방대하지만, 자주 쓰는 핵심 클래스만 알면 나머지는 공식 문서나 AI 도움으로 충분

<br/>

### 1-2. Tailwind CSS 설치

- 공식 문서: https://tailwindcss.com/docs/installation/using-vite

```bash
npm install tailwindcss @tailwindcss/vite
```

- **Vite 설정 파일에 추가하기**
  - `vite.config.ts` 파일에 tailwindcss 설정 추가

```javascript
import { defineConfig } from "vite";
import react from "@vitejs/plugin-react";
import tailwindcss from "@tailwindcss/vite";

export default defineConfig({
  plugins: [react(), tailwindcss()],
});
```

- **전역 CSS 파일에 추가하기**
  - `src/index.css` 파일에 tailwindcss 임포트하기

```css
@import "tailwindcss";
```

- **사용하기**

```javascript
function App() {
  return (
    <>
      <div className="font-bold underline">Hello World</div>
    </>
  );
}

export default App;
```

<br/>

### 1-3. Prettier 설치

- **Prettier 설치**

```bash
npm i -D prettier prettier-plugin-tailwindcss
```

- **Prettier 설정 파일 추가**
  - `.prettierrc` 파일 추가
  - 만약, VSCode가 prettier-plugin-tailwindcss 패키지의 엔트리 파일을 찾지 못하는 이슈가 있다면 직접 경로를 입력해준다.

```json
{
  // "plugins": ["./node_modules/prettier-plugin-tailwindcss/dist/index.mjs"]
  "plugins": ["prettier-plugin-tailwindcss"]
}
```

## 2. Tailwind CSS 핵심 정리

### 2-1. 타이포그래피

- **폰트 사이즈**
  - `text-xs`, `text-sm`, `text-base`, `text-lg`, `text-xl`, `text-2xl` ~ `text-9xl`로 텍스트 크기 설정
  - 동적 값 지정: `text-[13px]`처럼 대괄호 안에 원하는 값을 직접 입력
- **폰트 컬러**
  - `text-red-500`, `text-blue-300` 등 색상명 + 명도(50~950)로 텍스트 색상 설정
  - 명도 500이 원색에 가장 가까우며, 숫자가 높을수록 진한 색상
  - 동적 값 지정: `text-[rgb(100,30,200)]`처럼 직접 색상값 명시 가능
- **폰트 웨이트**
  - `font-thin(100)` ~ `font-black(900)`으로 텍스트 두께 설정
  - `font-bold`, `font-extrabold` 등 직관적인 이름으로 사용
- **배경 색상**
  - `bg-amber-500`, `bg-blue-400` 등 `bg-` 접두사 + 색상명 + 명도로 설정

<br/>

### 2-2. 사이즈

- Width & Height
  - `w-20`, `h-20` 등 숫자 × 기본 간격(spacing = 4px)으로 실제 크기 계산 (w-20 = 80px)
  - 대부분의 UI 디자인이 4px 단위를 기준으로 하기 때문에 이 방식을 채택
  - `w-[80px]`로 직접 픽셀 지정 가능, `w-full`로 부모 100% 너비 설정 가능
  - `min-w-`, `max-w-` 접두사로 최소/최대 너비도 설정 가능

<br/>

### 2-3. 여백

- **Padding**
  - `p-5` (전방향, 5×4=20px), `pt-5`(상), `pr-5`(우), `pb-5`(하), `pl-5`(좌)
  - `px-10` (좌우), `py-10` (상하)로 축 기준 설정 가능
- **Margin**
  - Padding과 동일한 방식으로 `m-`, `mt-`, `mr-`, `mb-`, `ml-`, `mx-`, `my-` 사용

<br/>

### 2-4. 보더

- **두께 (Border Width)**
  - `border`만 쓰면 1px 보더 자동 적용
  - `border-2`, `border-4` 등으로 두께 설정 (예외적으로 spacing이 곱해지지 않음, 숫자 = 그대로 px)
  - 방향 지정: `border-l`(좌), `border-r`(우), `border-t`(상), `border-b`(하), `border-x`, `border-y`
- **색상 (Border Color)**
  - `border-red-500`, `border-blue-300` 등 배경색과 동일한 방식으로 색상 설정
- **라운딩 (Border Radius)**
  - `rounded-xs` ~ `rounded-3xl` 등 텍스트 사이즈와 유사한 단계별 분류
  - `rounded-md`, `rounded-lg` 등으로 모서리 둥글기 설정

### 2-5. 플렉스 레이아웃

- 기본 설정
  - `flex` 클래스만 추가하면 자식 요소들이 가로(수평) 방향으로 배치됨
  - `flex-col` 추가 시 세로(수직) 방향으로 배치 전환
- 정렬
  - 수평 정렬: `justify-between`, `justify-center`, `justify-around` 등
  - 수직 정렬: `items-center`(중앙), `items-start`(상단), `items-end`(하단) 등

## 3. 예시

```javascript
function App() {
  return (
    <>
      <div>
        {/* 타이포그래피 */}
        <div className="text-xs">text-xs</div>
        <div className="text-sm">text-sm</div>
        <div className="text-lg">text-lg</div>
        <div className="text-xl">text-xl</div>
        <div className="text-2xl">text-2xl</div>

        {/* 백그라운드컬러 */}
        <div className="bg-amber-500">bg-amber-500</div>

        {/* 사이즈 */}
        <div className="h-20 w-20 bg-blue-500">box</div>
        <div className="h-[20px] w-[20px] bg-blue-500">box</div>
        <div className="w-full bg-blue-500">box</div>

        {/* 여백 */}
        <div className="h-50 w-50 bg-red-400 pt-5 pr-5 pb-5 pl-5">
          <div className="h-full w-full bg-blue-400"></div>
        </div>

        {/* 보더 */}
        <div className="m-5 rounded-e-md border-2 border-red-500">border</div>

        {/* 플렉스 컨테이너 */}
        <div className="flex flex-row items-start justify-between">
          <div className="h-10 w-10 border">1</div>
          <div className="h-20 w-10 flex-1 border">2</div>
          <div className="h-30 w-10 border">3</div>
          <div className="h-40 w-10 border">4</div>
        </div>
      </div>
    </>
  );
}
```
